### 삽입

***

###### 문제 설명

이분검색을 사용하면 정렬된 배열에 대하여 빠르게 검색이 가능할 뿐만 아니라 새로운 값의 삽입 위치도 빨리 알 수 있다.(삽입 위치 이후의 값들은 모두 한칸씩 뒤로 이동)


##### 제한 조건
x

###### 입출력 예

x |bs  |bs_left|bs_right|
  |:-- |:--    |:--     |:--
5 |0   |0      |0       | 
50|6   |6      |6       |
80|7   |7      |7       |
30|4   |4      |5       |
20|3   |1      |4       |
